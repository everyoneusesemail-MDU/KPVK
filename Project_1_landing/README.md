# Вилла Крым — Лендинг коттеджного посёлка

Современный редизайн сайта [villakrimkerch.ru](https://villakrimkerch.ru/) с акцентом на актуальные тренды веб-дизайна: анимации, 3D-эффекты, параллакс и цветовая схема по логотипу (изумрудно-зелёный и золотисто-бежевый).

## Запуск

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
npx serve .
# или
python -m http.server 8080
```

## Технологии

- **HTML5** — семантическая разметка
- **CSS3** — переменные (дизайн-токены), 3D transforms, анимации, параллакс-слои
- **JavaScript** — параллакс при скролле, 3D tilt карточки в hero, счётчики, AOS (Animate On Scroll), мобильное меню

## Цветовая палитра (из логотипа)

| Название        | HEX       | Использование                    |
|-----------------|-----------|----------------------------------|
| Emerald (основной) | `#2e5b53` | Фоны секций, акценты             |
| Gold / Beige    | `#c3a268` | Кнопки, заголовки, акценты       |
| Cream           | `#f8f5f0` | Светлый фон, текст на тёмном     |
| Sand            | `#e8e2d8` | Вторичный текст, границы         |

## Структура

- `index.html` — главная страница (hero, о посёлке, пакеты, инфраструктура, форма заявки)
- `css/styles.css` — стили и анимации
- `js/main.js` — параллакс, счётчики, 3D tilt, меню, AOS

## Особенности

- **Параллакс**: несколько фоновых слоёв двигаются с разной скоростью при скролле
- **3D-карточка в hero**: наведение мыши даёт лёгкий наклон (tilt)
- **Анимация по скроллу**: библиотека AOS для появления блоков
- **Счётчики**: анимированные цифры (5 мин, 10 мин, 969 домов) при появлении в зоне видимости
- **Адаптивность**: корректное отображение на мобильных, бургер-меню

# Вилла Крым — Приложение жителя

Статический MVP веб-приложения для коттеджного посёлка **«Вилла Крым»**.

## Что реализовано

- Авторизация по сгенерированному коду.
- Личный кабинет:
  - ФИО;
  - номер дома;
  - кадастровый номер;
  - номер участка;
  - модель и площадь дома;
  - задолженность и сумма оплаты услуг УК.
- Чат по веткам (ветки заданы владельцем приложения).
- Личные сообщения между домами.
- Сервисная служба:
  - описание проблемы;
  - срочность;
  - желаемая дата/время;
  - опция «Как можно скорее».
- Выписка гостевого пропуска:
  - ФИО и телефон гостя;
  - генерация отдельного QR на въезд и выезд;
  - таблица охраны (кто выписал, когда, отметки въезда/выезда).
- FAQ, Инструкция к дому (файлы/видео), Полезные номера.
- Новости посёлка и Афиша (демо-данные).

## Файлы

- `index.html` — структура интерфейса.
- `styles.css` — стили.
- `app.js` — логика приложения и демо-данные.

## Запуск

Откройте `index.html` в браузере.

## Демо-коды входа

- `VK-1042` (дом 42)
- `VK-1088` (дом 88)
- `VK-1120` (дом 120)

Также в форме входа есть кнопка «Сгенерировать демо-код».
